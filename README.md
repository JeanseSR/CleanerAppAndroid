# 🧹 CleanerApp

Application Android de nettoyage système — propre, rapide, sans pub.

![Build](https://github.com/JeanseSR/CleanerAppAndroid/actions/workflows/android.yml/badge.svg)

## Fonctionnalités

- 🗑️ Vider le cache des apps
- 📁 Supprimer les fichiers temporaires
- 🌐 Nettoyer les données navigateur
- 📦 Supprimer les APK téléchargés
- 🖼️ Nettoyer les miniatures médias
- 💬 Nettoyer WhatsApp / Telegram

## Stack

- Kotlin
- Jetpack Compose
- MVVM
- Min SDK : API 26 (Android 8.0)

## Build

```bash
./gradlew assembleDebug
```
